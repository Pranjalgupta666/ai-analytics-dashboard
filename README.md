## ✨ Features

- **Drag-and-drop CSV upload** with automatic column type detection (date / number / category / text)
- **Auto-generated dashboard** — KPI cards, line charts (time series), bar charts (category breakdowns)
- **AI executive summary** — headline, trends, anomalies, and recommendations generated by Gemini
- **Grounded Q&A chat** — ask questions about your data in plain English, answered from real computed stats, not guesses
- **Built-in sample dataset** — try it instantly with no upload required


## 🧠 Why this project

Most "AI dashboard" demos hand raw data straight to an LLM and hope it doesn't hallucinate. This one doesn't:

All the real math — sums, averages, min/max, time-series aggregates, category breakdowns — is computed in the browser first. Only that structured summary is sent to Gemini, which means:
- The AI's written summary can never disagree with the charts on screen
- Larger files don't blow past context limits or costs
- The same grounded data is reused for the chat feature, so follow-up answers stay consistent

## 🚀 Getting started

### 1. Clone and install

```bash
git clone https://github.com/<your-username>/ai-analytics-dashboard.git
cd ai-analytics-dashboard
npm install
```

### 2. Get a free Gemini API key

1. Go to [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Sign in with a Google account and generate a key — no credit card required
3. Copy it using the **Copy key** button

### 3. Set environment variables

```bash
cp .env.example .env.local
```

Paste your key into `.env.local`:

```
GEMINI_API_KEY=your_actual_key_here
```

### 4. Run it

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and click **"try it with sample sales data"**, or upload your own CSV.

## 🌐 Deployment

This app deploys cleanly to [Vercel](https://vercel.com) (built by the makers of Next.js, free tier available):

1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new) and import the repository
3. In the project's **Environment Variables** settings, add:
   - `GEMINI_API_KEY` = your key
   - `GEMINI_MODEL` = `gemini-2.5-flash` *(optional, this is the default)*
4. Deploy — Vercel auto-detects Next.js and handles the build

## 🔒 Security note

`GEMINI_API_KEY` is read only inside server-side API routes (`app/api/`) and is never sent to the browser. `.env.local` is excluded from git via `.gitignore` — never commit real API keys.
